# Traditional User Authentication System

## 📌 Project Description

This project is a **Traditional User Authentication System** built using HTML, CSS, JavaScript, PHP, and MySQL.  
It allows users to register, log in, and securely access protected areas of the website.

The system includes separate authentication files that handle user verification, login validation, and password security. Passwords are hashed before being stored in the database to improve security and protect user information.

This project demonstrates the implementation of a traditional authentication system used in many real-world web applications.

## 🔐 Authentication Process Overview

This system follows a secure authentication process:

1. User registers with email/username and password  
2. Password is hashed before storing in the database  
3. User logs in using credentials  
4. System verifies stored hashed password  
5. Session is created after successful login  
6. Protected pages are accessible only after authentication  
7. User can securely log out and session is destroyed
